# streamlit_dashboard

This is a simple streamlit dashboard focus on displaying financial data and informations allowing users to explore and viazualize some of the data.

Python libraries used to develop this aplication:

    Pandas
    Streamlit
    Plotly 
    Yfinance
    BeautifulSoup4
    requests 
    time
